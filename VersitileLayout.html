Content 1

This map displays geographical features represented by various icons such as theaters, bars, bicycles, and music venues. Each point is associated with an icon that corresponds to the type of feature it represents.

Content 2

There is a filter group located on the top right corner of the map which allows you to toggle the visibility of different types of features (theater, bar, bicycle, music venue). By checking or unchecking the checkboxes in the filter group, you can control which types of features are displayed on the map at any given time.
